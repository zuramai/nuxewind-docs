## Installation

Clone the template repository

<code-group>
  <code-block label="Yarn" active>

  ```bash
  git clone https://github.com/zuramai/nuxewind
  ```

  </code-block>
</code-group>


Then, install dependencies:

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn install
  ```

  </code-block>
  <code-block label="NPM" >

  ```bash
  npm install
  ```

  </code-block>
</code-group>

To run development, run:

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn dev
  ```

  </code-block>
  <code-block label="NPM" >

  ```bash
  npm run dev
  ```

  </code-block>
</code-group>

Then open `http://localhost:3000`
